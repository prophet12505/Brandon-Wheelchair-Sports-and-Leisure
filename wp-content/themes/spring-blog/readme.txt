== Description ==

Spring Blog is a clean and elegant WordPress theme built specifically for bloggers and blogging websites such as travel blog, gadget blog, magazine blog, personal blog, etc. It includes lots of features and can be customized very easily.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
